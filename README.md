# 🧠 CodexAILEN

Cognitive AI Simulation Engine with Memory and Reasoning System

---

## 🚀 Overview

CodexAILEN is a lightweight AI simulation engine designed to mimic cognitive behavior using:

- 🧠 Memory system (context retention)
- ⚙️ Behavior classification
- 🔎 Reasoning modes (question / deep / assist)
- 📊 Adaptive response state

This project is a browser-based AI simulation prototype, not a real LLM.

---

## 🌐 How it works

Input → Behavior Classifier → Memory Layer → Engine → Response Generator

---

## 🧠 Modes

- Normal Mode → basic response
- Question Mode → reasoning + memory scan
- Deep Mode → multi-layer analysis
- Assist Mode → step-by-step guidance

---

## 📂 Tech Stack

- HTML
- JavaScript
- LocalStorage (browser memory)

---

## 🚀 Run Project

Open `index.html` in any browser.
